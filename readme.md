***Environment***
Install
1. npm
2. node.js
3. python3
4. pip


Project Structure:

/frontend
/backend

***Build Frontend***
`cd frontend && npm i`

***Run Backend***
`cd backend && python3 app.py`


***AFTER BUILD FRONTEND AND HAVE BACKEND RUNNING***
Inside /frontend do `ng serve` and go to your Chrome browser: `http://localhost:4200`
